# Fine tuned Deepseek-r1 1.5B with Lora

This is an experimental repo for testing purpose. May not work as expected

## Install
- install pyenv virtualenv dependencies
```bash
$ pip3 install -r requirements.txt
```
- prepare `data.jsonl`

## Test base model
```bash
$ python run.py
```

## Train
```bash
$ python main.py
```

## Load model
```bash
$ python load.py
```
